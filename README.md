## ğŸš€ Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Pavansai20054/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## ğŸ“ Customization

### Personal Information
Update the following files with your information:
- `package.json` - Update author and description
- `components/Layout.jsx` - Update metadata
- `pages/index.jsx` - Update introduction and current focus
- `pages/contact/index.jsx` - Update contact information

### Projects
Edit `pages/projects/index.jsx` to add your projects with:
- Project title and description
- Tech stack used
- Live demo and GitHub links
- Detailed explanations

### Python Packages
Update `pages/packages/index.jsx` with your published packages:
- Package name and description
- PyPI badges
- Installation commands
- Usage examples

### Blog Posts
Modify `pages/blog/index.jsx` to include your blog posts:
- Post titles and excerpts
- Medium links
- Categories and tags
